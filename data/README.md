## Raw data

Place the BPI Challenge 2020 Travel Permit log here:

```
data/raw/PermitLog.xes
```

**Download:** https://data.4tu.nl/articles/dataset/BPI_Challenge_2020/12703980

The file is ~33 MB and is excluded from version control via `.gitignore`.

All notebooks expect the file at `data/raw/PermitLog.xes` relative to the project root.
